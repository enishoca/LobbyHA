# Changelog

## 0.5.0

- Initial Home Assistant add-on release
- Setup Wizard for first-run configuration
- Admin dashboard with area management, entity discovery, drag & drop reordering
- Guest dashboard — clean, read-only view for shared devices
- Guest PIN access — optional PIN gate for guest dashboard
- QR code access for guests
- WebSocket proxy for real-time entity updates
- SQLite storage — all config in a local database
- Configurable logging via add-on options
